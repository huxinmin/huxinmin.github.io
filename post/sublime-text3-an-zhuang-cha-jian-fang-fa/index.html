Sublime Text3安装插件方法

  1. 2020-04-09
  2. 310字
  3. 2 min read

1. 首先，安装package control

package control是sublime text的插件安装管理包，一般安装完sublime以后是不带这个的，需要自己手动安装，安装方法如下：

  1. 按Ctrl+`调出console（注：避免热键冲突）
  2. 粘贴以下代码到命令行并回车：
import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

2. 稍等一会，如果在Perferences->package settings中看到package control这一项，则安装成功。如果没有看到重启下sublime。

3. 使用package control安装软件的方法:

  • 安装插件
    摁下Ctrl(mac下为Cmd)+Shift+p，输入install，选择Package Control: install Package，摁下回车，即可输入插件名进行安装
  • 卸载插件
    摁下Ctrl(mac下为Cmd)+Shift+p，输入remove，选择Package Control: remove Package，摁下回车，选中插件名确认即可卸载

4. 常用的sublime插件推荐

  • Emmet（原名 Zen Coding）
    一种快速编写html/css的方法
    -Autoprefixer
    CSS3 私有前缀自动补全插件
  • HTML-CSS-JS Prettify
    使用说明：快速格式化html css js
    快捷键：ctrl+shift+h
  • ColorPicker
    需要输入颜色时可以自动出现调色板选取颜色
